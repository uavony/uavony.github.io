# Value and Unit ls（值与单位）

值的种类：
- 数字：`<number>` `<precent>`
- 颜色：`<color>` rgb等 十六进制颜色值#xxxxxx:rgb
- 函数：calc rgb rgba hsl hsla
- 图片：`<imags>`
- 位置：top left right center bottom

单位：
- 绝对长度单位
  - cm mm Q in pc pt px
- 相对长度单位
  - em：在font-size中使用相对于爸爸元素字体大小，其他元素使用相对与自己字体大小
  - ex
  - ch
  - rem
  - lh
  - vm
  - vh
  - vmin
  - vmax
